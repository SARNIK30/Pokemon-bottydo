Доступные команды

Команда Описание
/start Начать игру и выбрать стартового Покемона
/info Просмотр статистики тренера
/pokedex Просмотр Покедекса
/battle Вызвать другого тренера на битву
/shop Посетить магазин для покупки предметов
/promocode [код] Активировать промокод
/evolution [покемон] Эволюционировать Покемона (требуется 3 одинаковых Покемона)
/trade Обменяться Покемонами с другими тренерами
/admin Доступ к панели администратора (только для админов)

Работает на Pokemon API и Telegram Bot API
